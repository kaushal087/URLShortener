Documentation

URL Shortener

API URL = {{current_site}}shortenURL/
Method = POST
Return Type = JSON
Check working demo at {{current_site}}surl

Search

API URL = {{current_site}}search/?searchString=
Method = GET
Return Type = JSON
Check working demo at {{current_site}}search/?searchString=

Get no of Hits

API URL = {{current_site}}hits/?url= < URLSTRING >
Method = GET
Return Type = JSON
Check working demo at {{current_site}}hits/?url=p

URL redirection

visit {{current_site}}/url/< SHORT_URL_STRING >
It will redirect to the corresponding long URL
And if it is invalid the server will throw 404 Error.
Example: {{current_site}}/url/p
It will redirect to Google
